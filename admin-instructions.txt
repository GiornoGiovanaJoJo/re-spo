================================================================================
  ИНСТРУКЦИЯ: ПАНЕЛЬ УПРАВЛЕНИЯ RE-SPO (АДМИНКА)
================================================================================

ВХОД
-----
При первом сохранении или загрузке браузер может запросить логин и пароль
администратора. Данные запоминаются в браузере до выхода или очистки данных.


ОБЩЕЕ
-----
• Кнопка «Очистить кэш сайта» — сбрасывает кэш и перезагружает страницу;
  используйте после крупных правок, если на сайте не видно изменений.
• Кнопка «На сайт» — переход на главную страницу сайта.
• Блок «Последнее действие» показывает кратко, что было сделано последним.


ВКЛАДКА «МЕДИАФАЙЛЫ»
---------------------
• Слоты с фиксированными именами: загруженный файл сохраняется под именем
  слота и заменяет предыдущий файл с тем же именем.
• Поиск по имени файла — поле над списком.
• «Очистить все фото раздела» — удаляет изображения, относящиеся к этому
  разделу (действие с подтверждением).


ВКЛАДКА «ТОВАРЫ (КАТАЛОГ)»
----------------------------
КАТЕГОРИИ КАТАЛОГА
• Секции на странице /production и порядок блоков задаются списком категорий.
• ID категории: только латиница в нижнем регистре, цифры и символ подчёркивания,
  первый символ — буква. После появления товаров в категории ID лучше не менять.
• «+ Категория» — создание; в форме можно задать название, порядок сортировки,
  вид каталога (карусель / сетка / список), стиль карточки, колонки сетки,
  показ блока «Счётчик», какие поля товара использовать (описание, характеристики,
  таблица параметров, галерея на странице товара), подписи для страницы товара.
• «Сохранить на сервер» — записывает текущие категории и все товары в файл
  на сервере (без отдельного API только для категорий).
• Удалить категорию можно только если ни один товар на неё не ссылается.

ТОВАРЫ
• Добавление и редактирование — кнопки в таблице или «+ Добавить товар».
• Имя файла картинки: подставляется при загрузке из имени файла (кириллица
  и лишние символы обычно преобразуются). Файл с тем же именем на сервере
  будет заменён.
• Ссылка на страницу товара: при сохранении к /product при необходимости
  добавляется параметр id.
• Поля формы (описание, характеристики, параметры) могут скрываться в
  зависимости от выбранной категории — это соответствует настройкам категории.


ВКЛАДКА «СЕРТИФИКАТЫ»
---------------------
• Название, изображение (путь в assets/), текст описания.
• Загрузка изображения — по аналогии с товарами.


ВКЛАДКА «ОТЗЫВЫ»
-----------------
• На странице /production показывается ограниченное число отзывов (до 8).
• Редактирование текста и подписи автора.


ВКЛАДКА «ТЕКСТЫ»
----------------
• Редактирование site-text.json через API.
• На сайте подстановка: атрибут data-site-text="ключ" или
  data-site-text-placeholder для полей ввода.
• Фильтр по ключу, добавление нового ключа, «С сервера», «Сохранить все».


ВКЛАДКА «ГЛАВНАЯ»
------------------
• Четыре подписи плиток в блоке «Технологии» на главной странице.


ЕСЛИ ЧТО-ТО НЕ СОХРАНЯЕТСЯ
---------------------------
• Проверьте логин/пароль администратора.
• Посмотрите сообщение об ошибке (всплывающее уведомление).
• После правок попробуйте «Очистить кэш сайта».
